###### $ Dev. Setup

Ensure [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/download/) (which comes with [npm](http://npmjs.com)) are installed on your machine. From your command line:

```bash
# Clone this repo
$ git clone <repo link>

# Go into the repo
$ cd npm-dep-graph

# Install dependencies
$ npm i

# Run the dev-server
$ npm run dev
```
